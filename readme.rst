###################
Perdami untuk sisi User
###################

Project ini dibangun dengan Codeigniter 3 pada tahun 2018 untuk memudahkan resertifikasi dokter mata seluruh Indonesia di bawah assosiasi PERDAMI. Output akhir dari aplikasi ini adalah terbitnya sertifikat baru kepada dokter yang telah memenuhi syarat untuk mendapatkan sertifikat tersebut.
